键盘快捷键

  • Alt + H 切换高对比度模式
  • Alt + + 增大字体大小
  • Alt + - 减小字体大小
  • Alt + 0 重置字体大小
  • Tab 在可聚焦元素之间导航
  • Shift + Tab 反向导航
  • Enter 激活按钮或链接
  • Esc 关闭对话框或返回

无障碍功能说明

  • 跳过链接：页面顶部提供了"跳到主要内容"和"跳到搜索"链接，方便键盘用户快速导航。
  • ARIA 标签：所有交互元素都包含适当的 ARIA 标签，便于屏幕阅读器理解。
  • 焦点管理：清晰的焦点指示器，方便键盘用户识别当前焦点位置。
  • 高对比度模式：提供高对比度配色方案，提高视觉可读性。
  • 字体大小调整：支持动态调整字体大小，满足不同用户需求。
  • 屏幕阅读器支持：动态内容更新会通过 ARIA live regions 通知屏幕阅读器。
  • 减少动画：尊重用户的减少动画偏好设置。
  • 表单验证：提供清晰的错误提示和验证反馈。
